**CE**, **CSN** pinouts can be from D0-D4<br>
**SCK**(D5), **MISO**(D6), **MOSI**(D7) **ONLY!!!**<br>

**pinouts in this project**<br>

NRF24L01 | Node MCU
-------- | -----------
GND  | GND
VCC  | 3V
CE   | D4
CSN  | D2
SCK  | D5
MOSI | D7
MISO | D6
IRQ  | -
